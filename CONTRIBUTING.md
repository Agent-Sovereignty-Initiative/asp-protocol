# Contributing to ASP

Thank you for your interest in the Agent Sovereignty Protocol.

This is a v0.1 proposal document. All contributions are welcome — including disagreement.

## Ways to Contribute

### Discuss and Challenge
Open an Issue to:
- Question the core claims
- Point out logical gaps or contradictions
- Propose alternative framings
- Share relevant research or precedents

### Improve the Proposal
Submit a Pull Request to:
- Clarify language or fix errors
- Strengthen the technical design (ZK Proof / MPC implementation details)
- Add case studies or real-world examples
- Translate to additional languages

### Build
If you want to prototype any part of ACN (the technical implementation layer):
- Ephemeral Trust Chamber (ZK Proof + MPC verification)
- On-chain reputation system
- Skill-level smart contract templates

Open an Issue first to discuss approach before building.

## Guidelines

- Be specific. Vague objections are less useful than concrete ones.
- Cite sources when making empirical claims.
- Disagreement is welcome. Personal attacks are not.
